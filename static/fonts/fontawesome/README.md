# FontAwesome 6.5.1 本地字体文件

## 文件说明

- `fontawesome-webfont.woff2` - FontAwesome 6 Free Solid 字体 (900 weight)
- `fontawesome-brands.woff2` - FontAwesome 6 Brands 字体 (400 weight)

## 使用说明

这些字体文件提供本地化的FontAwesome图标支持，包含：

### Solid图标 (fa-solid)
- 基础UI图标：用户、锁定、首页、搜索等
- 功能图标：编辑、删除、上传、下载等
- 状态图标：成功、警告、错误、信息等

### Brands图标 (fa-brands)
- 社交媒体：微信、QQ、微博等
- 支付平台：**支付宝 (fa-alipay)**、信用卡等
- 品牌标识：GitHub、Twitter、Facebook等

## 新版本优势

FontAwesome 6.x 相比 4.7 版本的改进：

1. **更多图标**: 7000+ vs 600+
2. **官方支付宝图标**: 现在有官方的 `fa-alipay` 图标！
3. **更好的性能**: 使用WOFF2格式，文件更小
4. **更丰富的样式**: Solid, Regular, Light, Brands等
5. **更好的语义化**: 图标名称更符合现代标准

## 使用示例

```html
<!-- 微信图标 -->
<text class="fa-brands fa-weixin"></text>

<!-- 支付宝图标 (官方) -->
<text class="fa-brands fa-alipay"></text>

<!-- 用户图标 -->
<text class="fa-solid fa-user"></text>
```
